# Escuchito releases

Public release channel for Escuchito, a local transcription engine for macOS and Windows. This repo holds only installers, the update manifest, checksums, signatures, and install instructions. The source lives in a separate private repository.

The app checks this repo for updates. Every release is signed with the project's own updater key; the app verifies that signature before applying any update, so a compromised account cannot push a malicious update that installs silently.
